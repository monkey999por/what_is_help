Workflow
 - .github/workflowにあるファイル
 - ジョブを実行する構成ファイル
 - Eventでフックする
 
Event
 - ワークフローの実行をトリガーするリポジトリ内の特定のアクティビティ
   - ex. pull request

Job
 - 複数のステップをまとめたものをジョブという
 - 各ステップにはshell or Action(後述)で定義。ステップは上から順位実行される
 - 他のジョブとの依存関係を定義可能（Aが終わったらBをするとか）デフォルトはすべてのジョブが並列で走る

Action
 - 複雑だが頻繁に繰り返されるタスクを実行する GitHub Actions プラットフォームのカスタム アプリケーション
